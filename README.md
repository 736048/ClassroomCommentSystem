# Classroom Comment System

授業やプレゼンテーション、イベントで活用できる、リアルタイムコメント共有システムです。
参加者がスマートフォンやPCから送信したメッセージが、スクリーンの最前面を右から左へと流れていきます。

## 特徴

*   **リアルタイム・オーバーレイ表示**: 投稿されたコメントがデスクトップの最前面にオーバーレイ表示され、画面を流れます。
*   **スタイリッシュなデザイン**:
    *   **ブルータリズム風UI**: モノトーン（白・黒）を基調とした、角ばったシンプルで力強いデザインを採用。
    *   **レスポンシブ対応**: スマートフォン、タブレット、PCのいずれでも快適に操作できます。
*   **先生用管理機能 (コントローラー)**:
    *   **色付きコメント**: 先生のみ、色付きのコメントを強調送信できます（ブロック選択式）。
    *   **固定コメント**: 重要な連絡事項などを画面上部に固定表示できます。
    *   **NGワード設定**: 不適切な言葉をフィルタリングできます。
    *   **履歴管理**: 投稿されたコメントの履歴を確認できます（表示エリアは縦方向にサイズ調整可能）。
*   **生徒用機能**:
    *   **シンプル投稿**: 迷わず操作できるシンプルな投稿画面。
    *   **白文字固定**: 生徒のコメントは自動的に白文字で統一されます。
    *   **履歴表示**: 自分や他の人のコメント履歴を手元で確認できます。

## 必要条件

*   Node.js (推奨版)
*   npm

## インストール

プロジェクトのディレクトリで以下のコマンドを実行し、必要なパッケージをインストールしてください。

```bash
npm install
```

## 起動方法

以下のコマンドでアプリケーションを起動します。

```bash
npm start
```

## 使い方

1.  **起動**: アプリケーションが立ち上がると、サーバーが開始されます。
2.  **接続**: 
    *   管理画面に表示される **サーバーURL** (例: `http://192.168.1.x:3000`) を生徒（参加者）に共有します。
    *   生徒はそのURLにブラウザからアクセスします。
3.  **表示**:
    *   アプリケーションのウィンドウ背景は透明になります。この画面をプロジェクターやモニターに映し出してください。
    *   他のアプリケーション（スライド資料など）の上に重ねて表示することができます。
4.  **操作**:
    *   **生徒**: テキストを入力して「送信」ボタンを押します。
    *   **先生**: 別ウィンドウの「管理画面」からコメントの管理、強調コメントの送信、固定コメントの解除などを行います。

## ファイル構成

*   `main.js`: Electronのエントリーポイント
*   `server/`: サーバーサイド処理 (Socket.io)
*   `public/`: 生徒用画面 (HTML/CSS/JS)
*   `renderer/`: 先生用管理画面およびオーバーレイ画面 (HTML/CSS/JS)
