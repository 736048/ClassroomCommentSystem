# Classroom Comment System (ニコニコ風コメントアプリ)

授業やプレゼンテーションで使える、ニコニコ動画風のリアルタイムコメント配信システムです。
生徒がWeb画面から送信したコメントが、先生の画面（オーバーレイ）上に流れます。

## 主な機能

- **リアルタイムコメント配信**: 生徒がスマホやPCからコメントを送信すると、即座に画面上を流れます。
- **先生用コントロールパネル**:
    - **NGワード設定**: 不適切な単語をブロックできます（ポップアップなしで静かに更新）。
    - **先生コメント投稿**: 先生用の特別な色（白、赤、青、緑、黒から選択）でコメントを流せます。
    - **固定コメント機能**: 「固定表示」にチェックを入れると、コメントを画面上部中央に固定表示できます（解除ボタンあり）。
    - **コメント履歴**: 流れたコメントの履歴をリストで確認できます。
- **見やすいオーバーレイ**:
    - 上下に10%の余白を確保し、スライド等の邪魔になりにくい設計。
    - 太字＆縁取り文字で視認性抜群。

## インストール方法

Node.jsが必要です。

1. リポジトリをクローンまたはダウンロードします。
2. 依存関係をインストールします。
   ```bash
   npm install
   ```

## 使い方

1. アプリケーションを起動します。
   ```bash
   npm start
   ```
   （※ `package.json` に `start` スクリプトが定義されている前提。なければ `node main.js` 等）

2. 画面が表示されます。
   - **オーバーレイ画面**: `renderer/overlay.html` (またはアプリウィンドウ)。これをプロジェクターや共有画面に表示します。
   - **コントロール画面**: `renderer/control.html`。先生が手元で操作します。

3. 生徒に接続先を伝えます。
   - コントロール画面の「接続先URL」に表示されているURL（例: `http://192.168.1.5:3000`）を生徒に共有します。
   - QRコード生成ツールなどを使うとスムーズです。

4. 授業開始！
   - 生徒はブラウザからコメントを送信できます。
   - 先生はコントロール画面で流れを管理したり、重要なメッセージを固定表示したりできます。

## 設定

- **ポート番号**: デフォルトは 3000 です。変更したい場合は `main.js` または `server/app.js` を編集してください。
